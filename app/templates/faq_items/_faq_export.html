Project results can be downloaded as a zip archive from the status table, by clicking the download icon on the right. The archive contains the following files:

  • The original uploaded data, as a plain text file with no extension.
  • A tab-separated text file, with extension '.xls'. This is the main results file, which can be opened in Excel. Each row contains the following pathway attributes:
    • Full name
    • MSigDB pathway ID. These are used to name accompanying svg files
    • MSigDB info url
    • p-value
    • Effective pathway size (n_effective)
    • Actual pathway size (n_actual)
    • Gene mutation frequencies (%)
    • Shortest altered gene length (kbp)
    • Shortest altered gene name
    • Longest altered gene length (kbp)
    • Longest altered gene name
    • Average length of altered genes (kbp)
    • Variance in lengths of altered genes
  • A javascript file containing the basic pathway attributes used by results pages. This is simply a reformatting of the results text file.
  • Tree data, in 2 files. A compressed svg file (.svgz) contains the dendrogram plot. This can be opened in a web browser, such as Chrome. An accompanying text file (extension '.txt.reorder') contains the pathway names to label the branches, from top to bottom.
